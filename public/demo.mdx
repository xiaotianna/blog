# 文件名 Icon Demo

这个页面用于演示特殊文件名的代码块图标命中效果。每个示例都尽量保持精简，但会保证内容完整、可读。

## Language / File 图标

这一组主要用于覆盖 `fileStrategies` 的语言图标匹配，不依赖特殊文件名。
